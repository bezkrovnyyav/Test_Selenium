# Test_Selenium
Download  your version of Chromedriver   https://chromedriver.chromium.org/

Connect Chromedriver  to the PATH

pip install -Ur requirements.txt  --> installing dependencies

pytest test_mail.py  --> run the test

Test task:

При помощи python + selenium : регистрируем юзера на любом бесплатном сервисе электронной почты, отправляем сами себе письмо произвольного содержания, открываем свое письмо в папке "Входящие" и проверяем, что содержание совпадает с отправленным.
При написании теста придерживаться стандарта PEP-8
